## Project setup

1. Clone repo
```
git clone https://github.com/qwartzuiop/your-daily-dose-of-art.git
```

2. Install dependencies
```
npm install
```

3. Run development server
```
npm run serve
```

4. Compile and minify for production
```
npm run build
```
